# 族谱系统前端重设计方案

> 设计风格：中式古典/书法风格 - 传统水墨、毛笔质感、宣纸纹理
> 支持主题切换 | 全页面优化 | 移动端响应式

---

## 一、设计理念

### 1.1 核心美学
- **水墨意境**：以中国传统水墨画为灵感，运用墨色渐变、晕染效果
- **书法韵味**：标题采用书法字体风格，体现文化传承
- **宣纸质感**：背景融入宣纸纹理，营造古朴典雅氛围
- **留白之美**：遵循中式美学"计白当黑"，给予视觉呼吸空间

### 1.2 配色系统（三套主题）

#### 主题一：墨韵素雅（默认 - 浅色）

## 三、组件设计规范

### 3.1 卡片组件 (ZCard)
```
┌─────────────────────────────────────┐
│ ╭ 装饰性角纹 ╮                       │
│                                     │
│   【标题区】                         │
│   ──────────────────                │
│                                     │
│   内容区域                           │
│                                     │
│                       ╰ 装饰性角纹 ╯ │
└─────────────────────────────────────┘
特点：
- 半透明毛玻璃效果
- 微妙阴影（如水墨晕染）
- 四角可选装饰纹样
```

### 3.2 按钮组件 (ZButton)
| 类型 | 样式 |
|------|------|
| Primary | 朱砂红/紫檀棕背景，圆角矩形，微渐变 |
| Secondary | 透明背景，墨色边框，hover时填充 |
| Ghost | 无边框，仅文字，hover下划线效果 |
| Danger | 警戒红，用于删除操作 |

### 3.3 输入框组件 (ZInput)
- 底部单线边框（模拟毛笔勾勒）
- focus时线条加粗并变色
- 可选前缀图标（如小印章风格）

### 3.4 导航栏 (AppHeader)
```
┌──────────────────────────────────────────────────┐
│  🏛️ 族谱系统          成员 │ OCR │ 导出 │ 审计   │
│  [书法Logo]                          用户 ▼ 主题⚙ │
└──────────────────────────────────────────────────┘
移动端：汉堡菜单 + 侧滑抽屉
```

## 四、页面重设计

### 4.1 登录页 (LoginPage)
**布局变更**：
- 全屏背景：淡雅山水画渐变 + 宣纸纹理叠加
- 居中登录卡片：带传统纹样装饰
- 标题使用书法字体
- 印章风格的Logo装饰

**交互增强**：
- 输入框获得焦点时的墨迹动画
- 登录按钮点击涟漪效果
- 成功/失败的水墨风提示

### 4.2 成员管理页 (MembersPage)
**布局变更**：
- 顶部统计卡片（成员总数、在世人数等）
- 搜索栏重设计：融入书卷样式
- 成员列表改为卡片网格布局（支持列表/网格切换）
- 每个成员卡片显示头像占位符（圆形印章风格）

**新增功能建议**：
- 世代筛选标签
- 快速跳转到特定世代

### 4.3 成员详情页 (MemberDetailPage)
**布局变更**：
- 顶部大卡片：成员名（书法风格）+ 基本信息
- 关系图区域：可视化家谱连线（简易版）
- 信息区分组使用折叠面板
- 添加关系表单移至侧滑面板或模态框

### 4.4 OCR页面 (OcrPage)
**布局变更**：
- 拖拽上传区：书卷展开样式
- 任务状态用水墨风进度指示器
- 结果预览卡片化展示

### 4.5 导出页面 (ExportsPage)
**布局变更**：
- 导出选项卡片化选择
- 下载区域用卷轴展开动画
- 历史导出记录列表

### 4.6 审计日志页 (AuditsPage)
**布局变更**：
- 时间线样式展示（竖向卷轴感）
- 操作类型标签使用印章风格
- 详情弹窗使用传统卷轴样式

## 五、动效设计

### 5.1 过渡动画
- 页面切换：水墨晕染淡入淡出
- 卡片出现：从下方缓入 + 透明度渐变
- 列表项：依次延迟动画

### 5.2 交互反馈
- 按钮点击：墨点涟漪扩散
- 输入框focus：底线墨迹蔓延
- 删除操作：元素墨迹消散

### 5.3 加载状态
- 自定义水墨风 Loading 动画
- 骨架屏使用宣纸纹理
